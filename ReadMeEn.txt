// java version
Recommended JRE version - 19 (and newer)
https://www.java.com/en/download/

// how to install
The program uses Oracle software to work.
If you have set up an association with jar files, then double-clicking will launch the program.
Please download and install the latest recommended version of Java:
https://www.java.com/en/download/
Installation order:
https://www.java.com/en/download/help/windows_manual_download.html
How to make autostart on MS Windows boot:
https://support.microsoft.com/en-us/windows/add-an-app-to-run-automatically-at-startup-in-windows-10-150da165-dcd9-7230-517b-cf3c295d89dd

//Troubleshooting
If the program does not start:
You may need to set or change the PATH system variable:
https://www.java.com/en/download/help/path.html
Or reinstall Oracle JRE software:
https://docs.oracle.com/en/java/index.html
If the program does not open the .jar file (the same applies to MS Windows autorun):
Associate the jar files with the Java executable:
https://stackoverflow.com/questions/8511063/how-to-run-jar-file-by-double-click-on-windows-7-64-bit
